Résumé des Premiers Pas en R

Concept Détail / Explication Exemple Type
Création d'objets Utilisation de <- ou = pour assigner des valeurs à une variable. a <- 10 Variable simple
Vecteurs Vecteur : collection d'éléments. Utilisation de c() pour créer un vecteur. my_vector <- c(1, 2, 3, 4, 5) Vecteur numérique
Séquences Utilisation des deux-points : pour générer une séquence de nombres. my_seq <- 1:10 Vecteur entier (integer)
Matrices Structure de données bidimensionnelle avec matrix(). my_matrix <- matrix(1:9, nrow = 3, ncol = 3) Matrice
Fonction tail() Permet d'afficher les derniers éléments d'un vecteur, d'une matrice ou d'un autre objet. tail(my_vector) Fonction de manipulation
Accès aux éléments Accéder à un élément particulier d'un vecteur ou d'une colonne d'un data frame avec l'indexation. my_vector[1] ou data$colonne Accès à un élément
Type numeric Type de données par défaut pour les nombres (doubles). a <- 1.5 numeric ou double
Type integer Spécification explicite d'un entier en utilisant L ou la fonction as.integer(). a <- 5L ou b <- as.integer(3.7) integer
Conversion de types Utilisation de as.integer() pour convertir en entier, ou as.numeric() pour le contraire. a <- as.integer(3.7) Conversion de types
Fonction head() Afficher les premiers éléments d'un objet (par défaut 6 premiers). head(my_vector) Fonction de manipulation
Séquences avec L La notation L permet de créer directement des entiers sans conversion supplémentaire. a <- 5L integer
Manipulation de data.frame Accès à une colonne spécifique d'un data.frame via l'indexation ou $. data$colonne ou data[, 1] data.frame
Fonction typeof() Vérifie le type de données d'un objet. typeof(my_vector) Fonction de vérification
Fonction length() Renvoie la longueur d'un vecteur, d'une liste ou d'une matrice. length(my_vector) Fonction de vérification
